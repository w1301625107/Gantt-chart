# Gantt 图
## 有两个版本，基于Vue,React的

- Vue 版的文件为 base on vue.html
- React 版的文件为 base on react.html
  
分别使用不同的框架进行数据管理的Gantt图，通过jQueryUI来实现拖拽缩放功能,同时带有碰撞的控制功能
